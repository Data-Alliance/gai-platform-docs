# **GPU 공유 정보 확인**

노드 화면에서 현재 공유 중인 노드의 현황을 파악할 수 있습니다. <br><br>

![GPU공유정보_20251210_01.png](img/check-gpu-sharing-info/GPU공유정보_20251210_01.png)

1\. GPU 공유 기기가 한 대일 경우 위의 화면과 같이 내 공유 정보를 확인하실 수 있습니다.<br>
2\. 노드의 실행 상태와 노드명이 표시되며 노드명을 클릭 시 노드의 상세 정보를 확인할 수 있는 상세 페이지로 이동합니다.<br>
3\. GPU 스펙, 장치 스펙, 실행시간, 공유상태(실행/중지) , 총 수익 , 기본료, 사용료, 업데이트 일시 등의 정보를 확인하실 수 있습니다.<br><br>

![GPU공유정보_20251210_02.png](img/check-gpu-sharing-info/GPU공유정보_20251210_02.png)

4\. GPU 공유 기기가 여러 대일 경우 위와 같은 화면으로 표시되며, 각 GPU별로 정보 확인과 기능 실행이 가능합니다.<br><br>

![GPU공유정보_20251210_03.png](img/check-gpu-sharing-info/GPU공유정보_20251210_03.png)

5\. GPU 공유 기기의 갯수가 많은 경우 목록형 보기를 통해 한 눈에 요약 정보를 볼 수 있습니다.
