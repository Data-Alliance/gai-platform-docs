# **Check GPU Sharing Info**

On the Node screen, you can check the current status of the node you are sharing. <br><br>

![Checking_GPU_Sharing_Information_20260105_01.jpg](img.en/check-gpu-sharing-info/Checking_GPU_Sharing_Information_20260105_01.jpg)

1\. If you have only one GPU sharing device, you can check your sharing information as shown in the screen above.<br><br> 
2\. The **execution status** and **Node Name** will be displayed. Clicking on the node name will take you to a detailed page where you can view more specific information.<br> <br> 
- **GPU Specs & Device Specs:** Hardware model and technical specifications.<br> 
- **Uptime:** The total duration the node has been running.<br> 
- **Sharing Status:** Whether the node is currently "Running" or "Stopped."<br> 
- **Financial Data:** **Total Revenue**, **Base Fee**, and **Usage Fee**.<br> 
- **Timestamps:** The exact date and time of the last update.<br> <br> 

![Checking_GPU_Sharing_Information_20260105_02.jpg](img.en/check-gpu-sharing-info/Checking_GPU_Sharing_Information_20260105_02.jpg)

3\. If you have **multiple GPU sharing devices**, they will be displayed as shown in the screen above. You can view the specific information and execute functions for each individual GPU.<br><br>

![Checking_GPU_Sharing_Information_20260105_03.jpg](img.en/check-gpu-sharing-info/Checking_GPU_Sharing_Information_20260105_03.jpg)

4\. If you have a large number of GPU sharing devices, you can use the **List View** to see a summary of all information at a glance.
