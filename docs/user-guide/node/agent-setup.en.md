# **Agent Setup**

In the **Agent Setup**, you can update your **Graphics Drivers** and configure **VM (Virtual Machine) settings**. <br><br>

![Agent_Setting_20260105_01.PNG](img.en/agent-setting/Agent_Setting_20260105_01.PNG)

1\. Click on **"Setting."** <br><br>

![Agent_Setting_20260105_02.PNG](img.en/agent-setting/Agent_Setting_20260105_02.PNG)

2\. You can modify settings such as **reinstalling graphics drivers** or **deleting VMs**.<br>
