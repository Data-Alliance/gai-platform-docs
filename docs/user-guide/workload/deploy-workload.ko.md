# **워크로드 배포**

등록된 워크로드는 배포를 통해 자원 활용을 시작할 수 있습니다. <br><br>

![워크로드 배포_20251210_01.png](img/deploy-workload/워크로드%20배포_20251210_01.png)

1\. 워크로드에서 배포할 워크로드 항목의 “**배포**” 버튼을 클릭 후, “**확인**” 버튼을 클릭하세요. <br><br>

![워크로드 배포_20251210_02.png](img/deploy-workload/워크로드%20배포_20251210_02.png)

2\. 우측 상단의 워크로드 상태가 **미배포** → **배포** 상태로 변경되며, **서비스 URL**이 활성화 되는 것을 확인하실 수 있습니다. (몇 분정도 시간이 소요될 수 있습니다.)
