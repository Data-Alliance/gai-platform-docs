# **Workload Monitoring**

You can track and manage the resource usage of your registered workloads through the **"Monitoring"** feature.<br><br>

![Workload_Monitoring_20260106_01.jpg](img.en/Workload%20Monitoring/Workload_Monitoring_20260106_01.jpg)

1\. Click the **"Monitoring"** button located at the bottom right of the workload item. <br>

2\. A real-time monitoring dashboard will be displayed, allowing you to track the following metrics:<br>
- __GPU Utilization (%)__ <br>
- __GPU Memory Usage (MB)__  <br>
- __CPU Utilization (%)__  <br>
- __Memory Usage (MB)__  <br>
- __Network Inbound Packets (Kbps)__  <br>
- __Network Outbound Packets (Kbps)__  <br>
