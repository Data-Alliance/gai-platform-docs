# **Connect New Node**

To begin GPU sharing and earning rewards, you must first connect a new node to the platform. <br><br>

![Connect_New_Node_20260105_01.jpg](img.en/Connect-new-node/Connect_New_Node_20260105_01.jpg)

1\. In **Share Mode**, click on the **"Connect New Node"** menu in the left-hand sidebar.<br><br>

![Connect_New_Node_20260105_02.jpg](img.en/Connect-new-node/Connect_New_Node_20260105_02.jpg)

2\. Click the **"Download Program"** button to download the agent software.<br><br>

![Connect_New_Node_20260105_03.png](img.en/Connect-new-node/Connect_New_Node_20260105_03.png)

3\. Install the **agent program** on your system.<br><br>

![Connect_New_Node_20260105_04.jpg](img.en/Connect-new-node/Connect_New_Node_20260105_04.jpg)

4\. After downloading and running the agent, please follow the step-by-step instructions provided in the **Agent Execution menu** to proceed with the installation.<br><br>

![Connect_New_Node_20260105_05.png](img.en/Connect-new-node/Connect_New_Node_20260105_05.png)

5\. Once the installation is complete, **sharing will begin automatically**, as shown on the screen.
