# **Stop GPU Sharing**

You can stop GPU sharing through the **agent**. <br><br>

![Stop_GPU_Sharing_20260105_01.PNG](img.en/stop-gpu-sharing/Stop_GPU_Sharing_20260105_01.PNG)

1\. Click on the **"Running"** status in the center of the agent interface, then select **"Stop"** from the options **(Run / Stop)**.<br><br>

![Stop_GPU_Sharing_20260105_02.PNG](img.en/stop-gpu-sharing/Stop_GPU_Sharing_20260105_02.PNG)

2\. During the sharing suspension process, the status will change to **"Turning off"**, as shown in the image above.<br><br>

![Stop_GPU_Sharing_20260105_03.PNG](img.en/stop-gpu-sharing/Stop_GPU_Sharing_20260105_03.PNG)

3\. Once the sharing has been successfully stopped, the status will be displayed as **"Stopped."**<br>
