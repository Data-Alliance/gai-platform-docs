# **워크로드 종료**

워크로드는 ‘배포중지’ 버튼을 통해 사용을 중단할 수 있습니다.  <br><br>

![워크로드 종료_20251210_01.png](img/terminate-workload/워크로드%20종료_20251210_01.png)

1\. 워크로드 종료를 원하신다면, 배포된 항목의 “**배포중지**” 버튼을 클릭 후, “**확인**” 버튼을 클릭하세요. <br><br>

![워크로드 종료_20251210_02.png](img/terminate-workload/워크로드%20종료_20251210_02.png)

2\. 우측 상단의 워크로드 상태가 **배포** → **종료** 상태로 변경되며, 서비스가 중단됩니다. <br><br>

<aside>
⚠️<br>

종료 시, 백업 저장소를 설정하지 않은 경우 작업 중이던 데이터 및 환경은 <b>저장되지</b> 않으니 유의하시기 바랍니다. <br>
( 저장소 관리 > 백업 데이터 개인 저장소 연결 설정 후 워크로드 생성 시 개인 Storage 설정한 경우 데이터를 저장할 수 있습니다. )

</aside>
