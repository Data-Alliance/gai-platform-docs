# **Terminate Workload**

To stop a workload, click the **'Stop'** button. <br><br>

![Terminate_Workload_20260106_01.jpg](img.en/Terminate-Workload/Terminate_Workload_20260106_01.jpg)

1\.To terminate a workload, click the 'Stop' button on the deployed item, then click 'Confirm'. <br><br>

![Terminate_Workload_20260106_02.jpg](img.en/Terminate-Workload/Terminate_Workload_20260106_02.jpg)

2\. The workload status in the top right changes from **'Running'** to **'Finished'**, and the service is discontinued. <br><br>

<aside>
⚠️<br>

Please note that if a backup storage is not configured, your work data and environment will not be saved upon termination. (Data can be saved if you have configured personal storage settings when creating a workload after setting up the link in <b>Storage Management > Backup Data Personal Storage.</b>)

</aside>
